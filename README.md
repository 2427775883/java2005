"# 项目简介

## 核心功能
- **权限管理**：采用RBAC（基于角色的访问控制）实现管理员、酒店方和用户的分级权限管理
- **管理员功能**：审核酒店、管理用户、订单、公告等
- **酒店功能**：发布房间、管理订单和入住退房流程
- **用户功能**：在线预订、收藏、评价及查看入住记录

## 适用场景
- 酒店行业在线预订服务
- 多酒店统一管理平台
- 用户便捷预订和评价系统
- 酒店运营管理工具

## 技术特点
- 基于SpringBoot+Vue技术栈开发
- 采用RBAC权限控制机制
- 支持酒店分级审核流程
- 提供完整订单管理流程
- 用户评价和记录功能集成
